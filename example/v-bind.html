v-bind绑定样式

绑定一个或多个class,绑定一个style，绑定一个class表达式
单class绑定,v-bind:class="classA"
多class绑定,v-bind:class="[classA,classB]"
布尔型判断,:class="{classA:isA}"
三元表达式,:class="isA?classA:classB"
        var vue = new Vue({
            el: '#app',
            data: {
                classA: "classA",
                classB: "classB",
                isA:false,
                styleObject:{
                    fontSize:"12px",
                    fontColor:"#00f"
                }
            }
        })
